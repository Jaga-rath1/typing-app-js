# ⌨️ Typing App (JavaScript DOM Project)

## 📌 About

A simple typing app built using JavaScript DOM manipulation.
It captures user keyboard input in real-time and displays it on the screen.

## 🚀 Features

* Real-time typing display
* Supports all keyboard keys
* Space & Backspace handling
* Clean and minimal UI

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (DOM & Events)

## 🎯 What I Learned

* DOM Manipulation
* Event Listeners (keydown)
* Handling user keyboard input
* Basic UI structuring

## 📁 How to Run

1. Clone the repository
2. Open `index.html` in browser

## 🔥 Future Improvements
